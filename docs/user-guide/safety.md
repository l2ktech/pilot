# Safety

!!! danger "Important"
    Pilot controls a physical robot arm. Keep the workspace clear and stay alert during operation.

## Emergency Stop

### Stopping Motion

- Click the **STOP Robot** button (appears during motion)
- If your PAROL6 has a hardware e-stop, use it for immediate power cut

### Clearing E-Stop

1. Reset hardware e-stop if applicable
2. Click **Clear E-Stop** in the interface

---

## Live Control

!!! warning "Live Control Warning"
    With Live Control enabled, the robot moves immediately when you change positions.

**Before enabling:**

- Verify workspace is clear
- Set speed appropriately (start low)
- Be ready to stop motion

**Safe practices:**

- Disable Live Control when programming
- Preview motions with Live Control off first
- Keep movements small and deliberate
